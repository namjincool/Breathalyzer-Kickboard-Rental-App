
# Breathalyzer Kickboard Rental App (음주 방지 전동 킥보드 앱)

## Summary
이 프로젝트는 기존 킥보드 대여 서비스에 음주 측정 기능을 더하여,
음주 후 킥보드 대여를 제한함으로써 음주 운전 사고를 예방하는 데 목적을 두고 있습니다.
